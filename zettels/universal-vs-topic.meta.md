# Universal vs Topic Tag Groups

Tag groups have roles:
- **Universal** groups (Concern, Maturity) are expected on most zettels -- they answer fundamental questions every zettel should address ("what area?" and "how baked?")
- **Topic** groups (Foundations, Distribution, Data path) are optional clusters that help discovery for specific subsets of the knowledge base

The distinction is pragmatic, not structural. A zettel without a Concern tag isn't invalid -- it just might be harder to find. The roles guide usage without constraining it. Inspired by Gene Ontology's three universal aspects (Molecular Function, Biological Process, Cellular Component) which every gene annotation is expected to carry. This is the same [[tags-as-triples.meta]] mechanism applied at the group level.
